# Use Waze Data for Mobility Monitoring

## Lead Presenters

- Maria Sol Tadeo, World Bank Group (mtadeo@worldbank.org)
- Abril Rodriguez, World Bank Group (arodriguez12@worldbank.org)

For questions, suggestions, or feedback, please contact datapartnership@worldbank.org

## Training Description
In this session we will teach how to work with Waze Jams dataset. First, we will go over on how to perform a EDA on the data and present cases with anomalies. Secondly, we will show how to build maps of jams and the challenges of them.


## Audience and Prerequisites

- Intended audience: team/staff profile
- Prerequisites: domain knowledge, Python basics, GitHub basics
- Access requirements: dataset access, credentials, approvals

## Getting Started

1. Choose your setup path:
	- GitHub Codespaces (recommended): open the repository on GitHub and create a Codespace. No local clone is required.
	- Local environment: clone this repository to your machine.
2. Install dependencies.
3. Run the notebooks in the suggested order.

See [docs/github-codespaces-setup.md](docs/github-codespaces-setup.md) for a step-by-step Codespaces setup guide.
